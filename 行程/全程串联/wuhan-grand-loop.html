一、总览 + 决策逻辑

这是把 3 个独立行程串成一条 78 天大环线的总览页。3 个子行程都设计成"武汉进出"独立可跑；本页解释如何让它们衔接，省下中间的空跑和车托运。

  • 顺序：湘渝贵滇 → 东南亚 → 沿海五省。东南亚从云南进、回到云南（车寄存长水机场 20 天），不必飞回武汉再绕。
  • 核心衔接：① 湘渝贵滇 D22 在泸沽湖，不飞回武汉，而是 D23-D24 开车回大理→昆明寄存。② 东南亚 D1 改 昆明✈河内、D20 改 新加坡✈昆明。③ 取车后 D46-D49 走云南南部到广西，无缝接沿海五省（沿海 D1 改 南宁→桂林）。
  • 省下什么：丽江✈武汉 + 武汉→桂林 1,100km 两段空跑全砍。车一直在自己手里，无需托运 ¥3-5k。
  • 多拿什么：途中加 建水古城 + 元阳哈尼梯田 + 百色起义纪念馆 三个边境精华，单独跑很难安排。
✓ 子行程独立可跑：3 个原始 HTML（湘渝贵滇 / 东南亚 / 沿海五省）也支持单独跑（武汉进出），每段头尾都标注了"串联模式" vs "独立模式"两种选择。本总览页是串联模式的"粘合剂"。

六、关键决策 + FAQ

Q1：为什么不让东南亚作起点（先飞东南亚再开自驾）？

飞东南亚的最佳出发是季节性（11-3 月凉季最舒服）；自驾段对季节不敏感（中部 + 沿海全年可跑）。把东南亚放中段 等于把"季节锚点"放进总行程，更灵活。先开 22 天再飞，体力上也是"先紧后松"。

Q2：为什么不走中老铁路 / 河口陆路进东南亚？

① 飞机 2h 直达河内、价位 ¥800-1,500 跟陆路接近；② 陆路需 1-1.5 天，挤压东南亚 20 天；③ 中老铁路要把行程改成"老挝起步"（砍越南），跟原越泰马新顺序冲突。飞河内最干净。

Q3：车停长水机场 20 天，电瓶 / 轮胎怎么办？

电瓶：拔负极线（10mm 扳手）或带便携应急电源；轮胎：气压打到上限 +0.2bar（防慢漏）；油箱：半满最佳（防夏热膨胀 + 防油气挥发）；车内：清空食物 + 重要单据带走 + 关空调内循环。详见 衔接段 · 车寄存方案。

Q4：78 天太长，能压缩吗？

可以拆段或砍局部：

  • 砍东南亚 5 天（清迈/普吉/槟城任选一砍）→ 73 天
  • 砍珠三角 3 天（佛山+中山+东莞合并）→ 75 天
  • 砍涠洲岛（北海一日游）→ 77 天
  • 但不建议砍衔接段（建水/元阳/百色都是稀缺资源，且 4 天已经是地理最短）

Q5：D45 怎么算？

D45 是东南亚 D20 + 衔接段 B 的 D1 的合并日。新加坡早班机起飞，下午到昆明，傍晚取车后住机场酒店（如果体力够也能开 3h 到建水但不推荐）。所以日历上 D45 是 1 天，但既属于东南亚收尾又属于衔接段起点。

Q6：单独跑某段怎么用 3 个原文件？

3 个子 HTML 都保留了独立模式说明：

  • 湘渝贵滇：D22 后丽江飞武汉 + 车托运（原方案）
  • 东南亚：D1 武汉飞河内、D20 新加坡飞武汉（原方案，与昆明差价小）
  • 沿海五省：D1 改回 武汉→桂林 1,100km / 12h（原方案，建议拆 2 天）
